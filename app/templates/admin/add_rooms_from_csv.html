Format du fichier CSV

Le fichier CSV doit contenir les colonnes suivantes :

  • name - Nom de la salle (obligatoire)
  • capacity - Capacité de la salle (obligatoire, nombre entier)
  • location - Emplacement de la salle (obligatoire)
  • description - Description de la salle (optionnel)

Exemple de contenu CSV :

name,capacity,location,description
Salle A101,30,Bâtiment A - 1er étage,Salle de cours standard
Salle B205,15,Bâtiment B - 2ème étage,Salle de réunion
Amphithéâtre C,120,Bâtiment C - RDC,Grand amphithéâtre
Assurez-vous que votre fichier CSV est encodé en UTF-8 pour éviter les problèmes d'accents.
